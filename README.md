# Chat Bot: Hackathon Air Tahiti Nui

# Techno 

* HTML
* CSS
* Javascript (NodeJS)
* Wit.ai

# Use

Start client : 
```
cd client
ng serve
```

Start server : 
```
node bin/www
```

Test at : http://localhost:4200

**Note**:

For the moment, there is only some basics questions. Questions customers used to ask.
